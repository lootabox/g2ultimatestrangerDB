Welcome to the Ultimate Stranger Database

Something cool about this mod...

This database is adopted to be used as an interactive changelog for this mod. It has full statistics of all equipment, spells, consumables and monsters. New and changed features are described here, including all minor touches to the game.

Beyond those the other functionalities of the original database are disabled and hidden. This database is built on top of the downloadable version of Gothic II Addon Database site. All web design credit goes to authors of the original site.
